Como montar o sistema no Google Drive
1
Criar a estrutura de pastas no Drive
Crie uma pasta raiz chamada "BitLegacy" e subpastas conforme abaixo. Isso organiza tudo automaticamente.
📁 BitLegacy/ ├── 📁 01 - Reuniões/ │ ├── 📄 2026-04-14 - 1ª Reunião.html │ └── 📄 2026-04-14 - Transcrição.txt ├── 📁 02 - Documentos/ │ ├── 📄 Análise BitLegacy.html │ ├── 📄 Análise SovereignFolio.html │ └── 📄 Guia da Reunião.html ├── 📁 03 - Protótipos/ │ └── 📄 SentinelMail v0.1.html ├── 📁 04 - Pessoas/ │ └── 📄 Perfis dos participantes.html └── 📄 🏠 Project Hub.html ← este arquivo
2
Salvar cada arquivo gerado pelo Claude
Após cada sessão de trabalho, baixe os arquivos .html gerados e arraste para a pasta correta no Drive. Nomeie sempre com data: AAAA-MM-DD - Nome.html
3
Processar toda nova transcrição aqui
Depois de cada reunião, vá na aba "Processar call", cole a transcrição do Tactiq/Granola e clique em processar. O sistema extrai tudo automaticamente. Depois é só salvar o debriefing no Drive.
Fluxo completo: 1. Reunião acontece (Tactiq gravando) 2. Exporta transcrição do Tactiq 3. Cola aqui → clica "Processar com IA" 4. Revisa o resumo gerado 5. Baixa e salva no Drive: /01 - Reuniões/ 6. Atualiza tarefas na aba "Tarefas"
4
Compartilhar com a equipe (quando definir)
Quando definir quem entra no projeto, compartilhe a pasta "BitLegacy" no Drive com permissão de leitura. Para edição colaborativa, transforme os documentos principais em Google Docs.
5
Prompt padrão para processar transcrições
Use este prompt sempre que trouxer uma nova transcrição para o Claude:
Aqui está a transcrição da reunião do projeto BitLegacy. Por favor, extraia e organize: 1. DECISÕES TOMADAS — o que foi decidido formalmente 2. PRÓXIMOS PASSOS — quem faz o quê e até quando 3. IDEIAS NOVAS — conceitos ou funcionalidades sugeridos 4. DORES IDENTIFICADAS — problemas de mercado mencionados 5. PERFIL DOS PARTICIPANTES — o que cada um revelou sobre si 6. TOM DA REUNIÃO — como foi o engajamento geral Em seguida, gere: - Um debriefing visual completo (HTML) - As tarefas atualizadas do projeto - A pauta sugerida para a próxima reunião
